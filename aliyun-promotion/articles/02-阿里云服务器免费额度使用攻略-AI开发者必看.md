### 2. 开启Swap扩展内存
2核2G物理内存不够的情况下，可以开启Swap交换分区：
```bash
# 创建4G Swap文件
fallocate -l 4G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
```
这样总内存可以达到6G，足够运行7B量化模型。

### 3. 定时启停节省资源
对于非实时AI应用，可以设置定时任务，只在需要的时候启动：
```bash
# 每天8点启动，凌晨2点关闭
crontab -e
0 8 * * * systemctl start my-ai-app
0 2 * * * systemctl stop my-ai-app
```
这样可以减少资源占用，延长免费额度使用时间。
