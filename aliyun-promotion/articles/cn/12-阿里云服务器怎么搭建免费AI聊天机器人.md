## 二、搭建步骤（实测可行）

我们一步一步来，新手也能搞定：

### 步骤1：买好服务器，登录后台
1. 买完阿里云轻量应用服务器，选择CentOS或者Ubuntu系统，一键重装
2. 重置root密码，拿到服务器IP，用SSH登录上去

### 步骤2：开启Swap扩展内存（必须做）
2核2G物理内存不够跑模型，需要开Swap扩展：
```bash
# 创建4G Swap文件
fallocate -l 4G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
# 开机自动挂载，添加到/etc/fstab
echo '/swapfile none swap sw 0 0' >> /etc/fstab
```
做完这步，总内存就有6G，足够跑7B量化模型。

### 步骤3：安装运行环境
安装Git、Python、pip，这些都是基础环境，阿里云镜像已经有了，更新一下就行：
```bash
# Ubuntu/Debian
apt update && apt install git python3 python3-pip -y
# CentOS
yum update && yum install git python3 python3-pip -y
```

### 步骤4：下拉开源聊天项目，安装依赖
推荐用llama.cpp或者text-generation-webui，都是开源免费的：
```bash
git clone https://github.com/oobabooga/text-generation-webui.git
cd text-generation-webui
pip install -r requirements.txt
```
依赖安装需要几分钟，等待完成就行。

### 步骤5：下载量化模型，启动聊天
1. 从Hugging Face下载4比特量化的模型，比如Qwen-1.8B-Chat-GGUF
2. 放到模型目录，启动webui：
```bash
python server.py --model Qwen-1.8B-Chat-GGUF --gguf-precision q4_0
```
启动完成后，访问 `http://你的服务器IP:7860` 就能开始聊天了。
